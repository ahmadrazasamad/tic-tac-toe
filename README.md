# Tic Tac Toe Game

Welcome to the Tic Tac Toe game! This simple web-based game is implemented using HTML, CSS (TailwindCSS), and JavaScript. It supports both multiplayer and single-player modes.

## Features

- **Multiplayer Mode:** Play against a friend locally on the same device.
- **Single Player Mode:** Challenge yourself against an AI opponent.

## Technologies Used

- HTML
- CSS (TailwindCSS)
- JavaScript
